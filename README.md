# Steganography
Educational experience hiding messages into images.

## More detail
Read the [requirements](./docs/proposta_ufrn_bti_itp_2017_1.pdf).

## Setting up
In a Unix system, **c**hange **d**irectory to [./src/](./src/), and there, execute:

```sh
make && ./steganography.app
```
